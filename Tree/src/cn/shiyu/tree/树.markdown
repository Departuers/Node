### tree 
```
树的根结点为root

root没有父节点，空树高度为-1

没有左右子节点的称为叶子结点

结点层次level和树的深度Depth

层次从根开始定义，层次树为0的结点为根节点，其子树的层次树为1

高度height

树中结点的最大层次树被称为树的深度Depth或高度，树中结点也有高度，其高度为该结点为根的高度。

教科书上层数从0开始，写起来空结点高度为1

树的度Degree
结点拥有子树的数目成为结点的度
有几个孩子度就是几个

度为0的为叶子leaf结点。度不为0的结点称为非终端结点或分支结点。除树根之外的分支结点也称为内部结点。

没有孩子结点就是叶子结点leaf

树的性质
树中的结点数等于树的边数加1，也等于所有结点的度数之和加1
树开枝散叶，如果不看根节点，有多少条边就有多少个结点，最后加1
度同理，树根也是结点

树中的结点总数总是与边的总数相当，对于树的算法复杂性分析，可以用结点的数目作为规模的度量

路径
在树中通过k+1个结点通过k条边连接构成的序列，成为长度为k的路径path
比如连接4个结点，用三条边，这个路径

树中任意两个结点都存在唯一的路径，这意味着树连通又不坏路。从根节点开始，存在其他任意结点的一条唯一路径，根到某个路径的长度，恰好是该结点的层次数。
任何一个父节点都应该能找到它的子结点，有且只有一条路，

有序树，m叉树，森林forest

如果将树中结点的各子树看成从左到右是有次序的称之为有序树，不考虑子树顺序称之为无序树，一般都是讨论有序树

树中所有结点最大度数为m的有序树称之为m叉树

森林forest是m（m>=0）棵互不相交的树的集合，一棵树去掉根节点就是森林，加上树根，森林就变成一棵树。

属于逻辑结构，可以使用数组或者链式存储结构
