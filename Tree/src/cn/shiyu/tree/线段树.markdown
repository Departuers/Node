###线段树
````
经典的线段树问题：区间染色
有一面墙，长度为n，每次选择一块进行染色
m次操作后，我们可以看见多少种颜色
m次操作后，我们可以在【i，j】区间看见多少种颜色

实质：基于区间的统计查询
比如：2017年注册的用户消费最高的用户，
如果仅仅是2017年那一年的统计，直接查出来算就行了，
但数据在不断更新，就要用线段树

空间换时间
               使用数组实现   使用d线段树     
染色操作（更新区间）  O(N)      O(logn)
查询操作（查询区间）  O(N)      O(logn)

线段树其叶子节点可能不在同一层，

线段树和堆是一颗平衡二叉树，
把线段树看成一个满二叉树，不存在的节点看做null
用数组存储
